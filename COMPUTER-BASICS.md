## What is a Computer?

A **computer** is an electronic device that accepts data (input), processes it, and gives meaningful information as output.

It works on the principle of:
**Input → Process → Output → Storage**

---

## Main Parts of a Computer System

### 1. Hardware

Physical parts that you can touch:

* Monitor – Displays output
* Keyboard – Used to type
* Mouse – Used to click and select
* CPU (Central Processing Unit) – Brain of the computer
* Printer – Prints documents
* Speakers – Play sound
* Hard Disk / SSD – Stores data

---

### 2. Software

Programs that make hardware work:

**System Software**

* Operating System (Windows, Linux, MacOS, Android)

**Application Software**

* MS Word
* Excel
* Browser (Chrome)
* Media Player
* Photoshop

---

## Basic Units of Computer

| Unit        | Function                        |
| ----------- | ------------------------------- |
| Input Unit  | Takes data (Keyboard, Mouse)    |
| CPU         | Processes data                  |
| Memory      | Stores data                     |
| Output Unit | Gives result (Monitor, Printer) |

### CPU has three parts:

* **ALU** – Performs calculations
* **CU** – Controls all operations
* **Registers** – Fast memory

---

## Memory Types

### 1. Primary Memory

* RAM – Temporary memory (deleted when power off)
* ROM – Permanent memory

### 2. Secondary Memory

* Hard Disk
* SSD
* Pen Drive
* Memory Card
